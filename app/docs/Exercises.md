# Exercises

## Introduction

The Exercise Routes are used to

## `GET`

* Method: `GET`
* Endpoint: `{hostname}/{userID}/exercises/`

## `GET` by ID

* Method: `GET`
* Endpoint: `{hostname}/{userID}/exercises/{exerciseID}`

## `POST`

* Method: `POST`
* Endpoint: `{hostname}/{userID}/exercises/`

## `PUT`

* Method: `PUT`
* Endpoint: `{hostname}/{userID}/exercises/{exerciseID}`

## `DELETE`

* Method: `DELETE`
* Endpoint: `{hostname}/{userID}/exercises/{exerciseID}`
