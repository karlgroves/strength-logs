# Register, Confirm, Login, and Logout

## Introduction

## Register


* Method: `POST`
* Endpoint: `{hostname}/register/`

## Confirm

* Method: `GET`
* Endpoint: `{hostname}/confirm/{token}`

## Login

* Method: `POST`
* Endpoint: `{hostname}/login/`

## Logout

* Method: `POST`
* Endpoint: `{hostname}/logout/`
